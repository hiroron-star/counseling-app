# カウンセリング口コミ・求人情報サイト 仕様書

## 1. プロジェクト概要

### 1.1 目的

全国のカウンセリング店舗の検索、求人情報、相談掲示板機能を提供するWebアプリケーション

参考サイト: [カウンセリング.COM](https://www.kaunse-navi.com/counsel/kaunse)

### 1.2 スコープ（ミニマム機能）

初回リリースでは、以下のコア機能に絞って実装する：

- 店舗検索・一覧表示
- 店舗詳細表示
- 求人情報一覧・詳細表示
- 相談掲示板（投稿・閲覧）
- 専門家からの回答機能
- 会員の登録・ログイン機能（基本）

### 1.3 対象ユーザー

- **user（一般ユーザー）** - カウンセリングを探す、相談を投稿する
- **office（店舗運営者）** - 店舗情報を登録・管理、求人を掲載、相談に回答する

---

## 2. 機能一覧

### 2.1 user（一般ユーザー）向け機能

#### 店舗検索・閲覧
- エリア（都道府県・市区町村）で検索
- キーワード検索（店舗名、特徴など）
- 店舗一覧表示
- 店舗詳細表示（店舗情報、料金、営業時間など）

#### 相談掲示板
- 相談の投稿（タイトル、本文、都道府県、年代、性別）
- 相談一覧表示
- 相談詳細表示
- 専門家からの回答閲覧

#### 会員機能
- 会員登録
- ログイン・ログアウト
- 投稿した相談の管理

### 2.2 office（店舗運営者）向け機能

#### 店舗管理
- 店舗情報の登録
- 店舗情報の編集
- 店舗情報の削除（または非公開設定）
- 自分の店舗一覧表示

#### 求人情報管理
- 求人情報の投稿
- 求人情報の編集
- 求人情報の削除
- 自分の求人一覧表示

#### 相談への回答
- 相談への回答投稿
- 回答の編集・削除

### 2.3 共通機能

#### 求人情報閲覧
- 求人一覧表示
- 求人詳細表示（勤務地、給与、条件など）

## 3. 開発工程（フェーズ分け）

### フェーズ1: 基盤構築

**期間目安:** 1週間

1. **会員機能の実装**
   - user（一般ユーザー）の登録・ログイン
   - office（店舗運営者）の登録・ログイン
   - ログアウト機能

2. **基本的な画面構成**
   - トップページ
   - ヘッダー・フッター
   - 共通レイアウト

### フェーズ2: 店舗関連機能

**期間目安:** 2週間

1. **店舗登録・管理機能（office向け）**
   - 店舗情報の登録フォーム
   - 店舗情報の編集
   - 店舗情報の削除
   - 自分の店舗一覧表示

2. **店舗検索・表示機能（user向け）**
   - 店舗一覧表示
   - エリア検索（都道府県で絞り込み）
   - キーワード検索
   - 店舗詳細表示

### フェーズ3: 相談掲示板機能

**期間目安:** 2週間

1. **相談投稿機能（user向け）**
   - 相談投稿フォーム
   - 相談の保存
   - 自分の相談一覧表示

2. **相談閲覧機能**
   - 相談一覧表示
   - 相談詳細表示
   - 都道府県で絞り込み

3. **回答機能（office向け）**
   - 相談への回答投稿
   - 回答の編集・削除
   - 回答済み相談の管理

### フェーズ4: 求人情報機能

**期間目安:** 1.5週間

1. **求人情報登録・管理機能（office向け）**
   - 求人情報の投稿フォーム
   - 求人情報の編集
   - 求人情報の削除
   - 自分の求人一覧表示

2. **求人情報閲覧機能**
   - 求人一覧表示
   - 求人詳細表示

### フェーズ5: UI/UX改善・テスト

**期間目安:** 1週間

1. **デザインの統一**
   - 画面の見た目の調整
   - スマホ・タブレット対応
   - エラーメッセージの改善

2. **動作テスト**
   - 各機能の動作確認
   - バグ修正

3. **リリース準備**
   - 本番環境への公開
   - 最終チェック

## 7. 用語集

- **user** - 一般ユーザー（カウンセリングを探す、相談を投稿する人）
- **office** - 店舗運営者（店舗情報を登録・管理、求人を掲載、相談に回答する人）
- **店舗** - カウンセリングを提供する事業所
- **相談** - userが投稿する相談内容
- **回答** - officeが相談に対して返す回答
- **求人** - カウンセラー募集の求人情報
- **MVP** - Minimum Viable Product（最小実行可能製品）
